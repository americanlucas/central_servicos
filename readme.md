# Central de Serviços

Sistema para centralização e gestão de serviços internos.

## Descrição

Este projeto é uma aplicação web desenvolvida para gerenciar e centralizar diferentes serviços internos em uma única plataforma, facilitando o acesso e a administração dos recursos disponíveis.

## Funcionalidades

- Centralização de serviços em um único painel
- Interface intuitiva e responsiva
- Gestão de acessos e permissões
- Integração com sistemas internos

## Tecnologias Utilizadas

- Flask
- SQLAlchemy
- HTML e CSS
- JavaScript

## Instalação

```bash
# Clone o repositório
git clone https://github.com/americanlucas/central_servicos.git

# Entre no diretório
cd central_servicos

# Crie um ambiente virtual (recomendado)
python -m venv venv

```
